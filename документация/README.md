# Настройка Ubuntu для разных направлений разработки

Русскоязычный справочник команд для подготовки Ubuntu под учебные и демонстрационные задачи: от веб-разработки и DevOps до IoT, компьютерного зрения и промышленных систем.

## Как пользоваться

1. Откройте [базовые настройки](документация/00-базовые-настройки.md) и выполните только необходимые общие шаги.
2. Выберите один вариант из таблицы ниже.
3. Прочитайте команды целиком до запуска и адаптируйте версии, имена пользователей и пути под свою систему.
4. После установки выполните [проверку](документация/проверка-выполнения.md).
5. При необходимости заполните [шаблон руководства пользователя](документация/шаблон-руководства-пользователя.md).

> [!CAUTION]
> Многие команды запускаются через `sudo`, устанавливают ПО, создают пользователей или меняют системные службы. Не вставляйте весь файл в терминал одним блоком. Учебные пароли из примеров обязательно замените. Команды не тестировались как единый автоматический установщик, а внешние URL и версии со временем могут устареть.
