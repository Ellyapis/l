# Вариант 28: Биометрические системы (Python, компьютерное зрение)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА PYTHON И OPENCV
# =====================================================
sudo apt install -y python3 python3-pip
pip3 install opencv-python

# =====================================================
# УСТАНОВКА DLIB (распознавание лиц)
# =====================================================
sudo apt install -y libdlib-dev
pip3 install dlib

# =====================================================
# УСТАНОВКА FACE RECOGNITION
# =====================================================
pip3 install face-recognition

# =====================================================
# УСТАНОВКА CUDA (для GPU)
# =====================================================
sudo apt install -y nvidia-cuda-toolkit

# =====================================================
# УСТАНОВКА V4L2 (камеры)
# =====================================================
sudo apt install -y v4l-utils
```
