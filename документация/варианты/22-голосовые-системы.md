# Вариант 22: Голосовые системы (Python, аудиообработка)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА PYTHON И АУДИО-БИБЛИОТЕК
# =====================================================
sudo apt install -y python3 python3-pip portaudio19-dev
pip3 install librosa sounddevice pyaudio

# =====================================================
# УСТАНОВКА VOSK (голосовой ассистент)
# =====================================================
pip3 install vosk

# =====================================================
# УСТАНОВКА WHISPER (OpenAI)
# =====================================================
pip3 install openai-whisper

# =====================================================
# УСТАНОВКА ALSA (звук)
# =====================================================
sudo apt install -y alsa-utils
sudo alsactl init
```
