# Вариант 27: SCADA для энергетики (C++, IEC 61850, DNP3)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА C++ КОМПИЛЯТОРА
# =====================================================
sudo apt install -y build-essential cmake

# =====================================================
# УСТАНОВКА LIBIEC61850
# =====================================================
git clone https://github.com/mz-automation/libiec61850
cd libiec61850
make
sudo make install

# =====================================================
# УСТАНОВКА OpenDNP3
# =====================================================
git clone https://github.com/automatak/dnp3
cd dnp3
mkdir build && cd build
cmake ..
make
sudo make install

# =====================================================
# УСТАНОВКА QEMU (для эмуляции QNX)
# =====================================================
sudo apt install -y qemu-system-x86_64

# =====================================================
# УСТАНОВКА WIRESHARK
# =====================================================
sudo apt install -y wireshark
```
