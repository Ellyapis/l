# Вариант 11: Микроконтроллеры (STM32)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА ARM TOOLCHAIN
# =====================================================
sudo apt install -y gcc-arm-none-eabi binutils-arm-none-eabi gdb-arm-none-eabi

# =====================================================
# УСТАНОВКА OPENOCD (JTAG/SWD)
# =====================================================
sudo apt install -y openocd

# =====================================================
# УСТАНОВКА QEMU (ARM эмулятор)
# =====================================================
sudo apt install -y qemu-system-arm qemu-system-aarch64

# =====================================================
# УСТАНОВКА PLATFORMIO
# =====================================================
pip3 install platformio
```
