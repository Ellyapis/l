## 4. Плановый выход из программы

1. Сохранить все открытые документы или проекты, если требуется.
2. Выбрать в меню **File → Exit** (или **Файл → Выход**).
3. Либо закрыть окно программы стандартной кнопкой закрытия.
4. Дождаться полного завершения всех процессов.

> Если программа запущена в интерактивном режиме через терминал, для её остановки обычно используется `Ctrl+C`.
