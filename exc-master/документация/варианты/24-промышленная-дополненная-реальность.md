# Вариант 24: Промышленные AR-решения (Unity, AR-гарнитуры)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА UNITY HUB
# =====================================================
wget -O unityhub.deb https://public-cdn.cloud.unity3d.com/hub/prod/UnityHubSetup.deb
sudo dpkg -i unityhub.deb 2>/dev/null || sudo apt --fix-broken install -y
sudo dpkg -i unityhub.deb

# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА BLENDER
# =====================================================
sudo snap install blender --classic

# =====================================================
# УСТАНОВКА ANDROID STUDIO (для AR-гарнитур)
# =====================================================
sudo snap install android-studio --classic

# =====================================================
# КЛОНИРОВАНИЕ MRTK (Mixed Reality Toolkit)
# =====================================================
git clone https://github.com/microsoft/MixedRealityToolkit-Unity
```
