# Вариант 16: Медицинское ПО (C++, Python, HIPAA/GDPR)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА CLion (C++ IDE)
# =====================================================
sudo snap install clion --classic

# =====================================================
# УСТАНОВКА PYTHON
# =====================================================
sudo apt install -y python3 python3-pip python3-venv

# =====================================================
# УСТАНОВКА POSTGRESQL С КРИПТОГРАФИЕЙ
# =====================================================
sudo apt install -y postgresql postgresql-contrib
sudo systemctl start postgresql
sudo systemctl enable postgresql

# =====================================================
# УСТАНОВКА DICOM-ПРОСМОТРЩИКА (медицинские изображения)
# =====================================================
sudo apt install -y aeskulap

# =====================================================
# УСТАНОВКА R (статистика)
# =====================================================
sudo apt install -y r-base

# =====================================================
# УСТАНОВКА КРИПТОГРАФИЧЕСКИХ БИБЛИОТЕК
# =====================================================
pip3 install cryptography pycryptodome pandas numpy
```
