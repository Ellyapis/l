# Вариант 21: CAD/CAM системы (C++, 3D-графика)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА FREECAD (параметрическое моделирование)
# =====================================================
sudo snap install freecad

# =====================================================
# УСТАНОВКА OPENGL БИБЛИОТЕК
# =====================================================
sudo apt install -y libglu1-mesa-dev freeglut3-dev mesa-common-dev

# =====================================================
# УСТАНОВКА BLENDER
# =====================================================
sudo snap install blender --classic

# =====================================================
# УСТАНОВКА GIT LFS (для больших файлов)
# =====================================================
sudo apt install -y git-lfs
git lfs install

# =====================================================
# УСТАНОВКА OpenCASCADE (геометрическое ядро)
# =====================================================
sudo apt install -y liboce-ocaf-dev liboce-ocaf-lite-dev
```
