# Вариант 8: SPA-приложения (React/Angular/Vue)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА NODE.JS (LTS)
# =====================================================
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# =====================================================
# УСТАНОВКА ФРЕЙМВОРКОВ
# =====================================================
sudo npm install -g @angular/cli
sudo npm install -g @vue/cli
sudo npm install -g create-react-app vite

# =====================================================
# УСТАНОВКА MONGODB
# =====================================================
sudo apt install -y mongodb
sudo systemctl start mongodb
sudo systemctl enable mongodb
```
