# Вариант 26: Чат-боты (Python, NLP, Dialogflow)

> Команды собраны из исходного материала без проверки на конкретной версии Ubuntu. Версии пакетов, ссылки и доступность Snap могут меняться.

```bash
# =====================================================
# УСТАНОВКА VS CODE
# =====================================================
sudo snap install --classic code

# =====================================================
# УСТАНОВКА PYTHON И NLP-БИБЛИОТЕК
# =====================================================
sudo apt install -y python3 python3-pip
pip3 install nltk spacy transformers

# =====================================================
# УСТАНОВКА SPACY МОДЕЛИ (РУССКИЙ ЯЗЫК)
# =====================================================
python3 -m spacy download ru_core_news_sm

# =====================================================
# УСТАНОВКА RASA (фреймворк диалогов)
# =====================================================
pip3 install rasa

# =====================================================
# УСТАНОВКА TELEGRAM BOT API
# =====================================================
pip3 install python-telegram-bot

# =====================================================
# УСТАНОВКА REDIS (кеш)
# =====================================================
sudo apt install -y redis-server
sudo systemctl start redis-server
```
